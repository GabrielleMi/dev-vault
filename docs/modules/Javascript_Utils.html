    Utility functions

    Remarks

    This library is an experimental project.

    A collection of utility functions and constants designed for personal projects. This library is an experimental project provided as-is.

    Installation

    pnpm add @dev-vault/utils

    Features

    • Array: Operations like splitInChunks, toArray, and safeArray.
    • Constants: HTTP codes, time units, and common primitive type identifiers.
    • CSP: Tools for generating Content Security Policy headers.
    • Type Checking: Robust helpers for primitive types, objects, and null checks.
    • JSON: Safe stringify and parse implementations.
    • Object/Number: Tools for deep property access and safe division.
    • Search/Sort: Configurable searching and sorting utilities for objects and arrays.
    • String: Utilities for case-insensitive comparison and safe joining.
    • Time: Time unit conversion helpers.
